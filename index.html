About Vibe Coding

Coding with creativity, curiosity, and AI

Vibe coding is about taking an idea and bringing it to life with code, even if you don’t know all the details of a programming language. AI can help write the code, explain concepts, and fill in the gaps so the focus stays on the idea itself.

On this site, I share projects I’ve vibe coded. It’s a place to show what can be done when you mix creativity, curiosity, and coding support from AI. Each project is part of my journey exploring what’s possible.

Vibe coding is about energy, experimentation, and expression. It’s not just technical — it’s creative. This page is where I put that into practice.
